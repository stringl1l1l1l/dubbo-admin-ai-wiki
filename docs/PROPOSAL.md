# Proposal 模板
一份好的 Proposal，目标不是把内容写长，而是让后来者能快速回答三件事：

1. 当时为什么要做这件事
2. 最终为什么选了这个方案
3. 这个方案带来了哪些新约束和后续任务
4. 应该如何实现这个功能

## 1. 背景

说明当前问题、业务上下文、触发原因和已有痛点。

建议回答：

- 问题出现在哪里
- 为什么现在必须处理
- 如果不处理，会带来什么后果
